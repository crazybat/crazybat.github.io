Get Started

Ready to start using GCbootstrap?

Follow these steps to get you started with a basic template that shows you most of the basic Bootstrap styles.

Experience with HTML, CSS, JS and GCpedia recommended!

Last updated: 2018-12-11

Important!

Unfortunately, I am unable to personally assist you with the creation and editing of your pages. However, there is a current GCconnex support group where you can post your questions: GCconnex: GCbootstrap

As more users start using GCbootstrap, we can utilize the growing knowledge by creating specific community groups to assist others. In the meantime, if you need guidance to get started, let me know and I'll do my best to help.

Create the GCpedia Page

  • Open a new tab or window in your browser
  • Go to GCpedia and click Log in on the top-right of the page
    • No account? Follow the instructions on creating one.
  • Using the Search GCpedia box (not Search GCTools at the top right), type in the name of the page you want to create.
    • If your name is taken, keep trying another name until no search results are found.
  • Important! Once your new page is created, copy the current URL and save it somewhere (Notepad, Word, etc.). This will be the new location of your page and you'll need to edit it later on.

Paste the Code

  • Navigate to your new page and click Edit Source
  • Paste the code from Step 1.
  • Click Save Page
  • If successful, your new page should look exactly like the GCbootstrap Style Guide.
  • To edit your new page, the Edit Source link is now located in the GCpedia Tools menu on the bottom right of the screen.

Important! Review the Files

By default, your new page is now using the default settings, templates, or include files.

Make sure to take the following action(s) before you start to edit:

File Reference Description Knowledge Required Action
__TOC__ (English / French) Removes the default GCpedia Table of Contents. GCpedia If you want to keep a Table of Contents on your page, simply remove this line.
GCb-head-yoursite-eng (English) Applies all of the required Bootstrap files for proper styling. GCpedia HTML CSS JS Make a copy. Don't edit this file! Then, point to the new file on your template.
GCb-nav-yoursite-eng (English) Site Navigation file that populates the main navigation. GCpedia HTML Make a copy. Don't edit this file! Then, point to the new file on your template.
GCb-footer-yoursite-eng (English) Site Footer file that populates the bottom of your site. GCpedia HTML JS Make a copy. Don't edit this file! Then, point to the new file on your template.
GCb-css-yoursite-core (English / French) Core styling for your site. GCpedia CSS Make a copy. Don't edit this file! Then, point to the new file on your template.
(Note: If you want to use only the default styles, then it's highly recommended you keep this reference.)
GCb-css-yoursite-override (English / French) Stying that brings back GCpedia's menus and functionalities. GCpedia CSS Make a copy. Don't edit this file! Then, point to the new file on your template.

If there are styles that you would like to add that aren't part of these files, you can create a reference to your own CSS file on your template. To make sure your styles don't conflict with GCbootstrap, place the reference at the very last line.
