Welcome to GCdocs

As employees of the Government of Canada, we are constantly surrounded with a large amount of information, whether it is e-mails, letters, information for the media, reports, etc. It is this information that allows the majority of employees to continue to successfully do our jobs.

As public servants, it is our duty to manage information properly so it’s easily accessible to whoever requires it. Our colleagues, departments and the population of Canada rely on public servants to organize and manage all information in a responsible and efficient manner. That way, when it is needed, it is at one’s disposal.

GCdocs allows for all created information to be stored in a single, secure and organized location. The system allows public servants to share, use and access information across the organization, ensuring it is there for our convenience.

This guide to GCdocs will help answer any questions you may have as a new user of GCdocs.

What is GCdocs?

GCdocs is the Government of Canada’s official Electronic Document Records Management Solution (EDRMS). It is an automated and bilingual archiving and information management system whose functions can be adjusted to meet organizations’ requirements. GCdocs helps different organizations meet their obligations in relation to information life cycle management.

This tool helps organizations create, collect and preserve data in an electronic environment, as well as easily sharing, organizing, evaluating, identifying and disposing of data.

While encouraging group collaboration, it also allows for secure access control through the management of user and group rights.

GCdocs also supports the open information stream of the Government of Canada Open Government Initiative. It is strategically positioned to address the challenge of effectively managing public resources, in addition to increasing public integrity and improving public services.

Changing the Language

To change the login page to French

  1. Select Français (Canada) located at the bottom of the login page.

To change the language within GCdocs

  1. In the menu bar, click Tools.
  2. In the drop-down menu, click Settings.
  3. This will bring you to the General Tab of your General Settings.
  4. On the right-hand side, click Language.
    1. Click English or French.
    2. This will change the system’s language.
  5. On the right-side, click Metadata Language.
    1. Click English or French.
    2. This will change the name of the folders and documents to French, if a French name was entered for the document or folder using the multilingual settings.

Saving Documents

To save a new document directly to a folder

  1. Create a new document in Microsoft Office (Word, Excel, PowerPoint, etc.).
  2. From the Office top bar, select Enterprise Connect > Save As.
  3. Browse the Directory tree and select the Folder you want to save the document to.
  4. Type a name for the document in the Name field. Click Save.

To add an item with Drag and Drop

  1. Click an item on your desktop and drag it into the Drag and Drop area.
  2. Optionally, you can drag and drop from a folder or from an E-mail.

To add a version to an existing GCdocs document

  1. Click the item’s Functions icon, and then choose Add Version.
  2. Click the Browse button, navigate to the file, and then click the Open button.
  3. Click the Add Version button.

To send and save

  1. Create a new E-mail in Microsoft Outlook.
  2. Click the Send and Save button.
  3. Browse the Directory Tree and select the folder you want to save the message to.
  4. Click Save.

To file an E-mail with Drag and Drop

  1. Expand the Enterprise Connect folder in Microsoft Outlook to the folder where you want to add an E-mail message.
  2. Click an item in your Outlook folder and drag it to the appropriate folder in GCdocs.

To add a Folder in GCdocs (Permissions required)

  1. Click the Add Folder button.
  2. Type a name for the folder in the Name field.
  3. Click Add.

To add an E-mail Folder in GCdocs (Permissions required)

  1. Click the Add Item > E-mail Folder button.
  2. Type a name for the folder in the Name field.
  3. Click Add.

Changing Permissions

NOTE: By default, all items added to GCdocs are accessible to all DFO / Canadian Coast Guard employees. Restricted when necessary.

To change Permissions for individual items

  1. Click the item’s Functions icon and choose Permissions.
  2. On the Permissions page, click a user or group’s link in the Default Access or Assigned Access list.
  3. In the Edit Permissions section for the user or group, modify the permissions settings, and then click the Update button.
  4. Click the Done button.

NOTE: This assumes you have permission to make the change.

Sharing Information by E-mail

To share documents by E-mail - From GCdocs

  1. Locate the document in GCdocs.
  2. Select the Checkbox to the left of the file name.
  3. Click the E-mail Link button

To share documents by E-mail - From Enterprise Connect

  1. Locate the document in GCdocs.
  2. Right-click on the file and choose Send To.
  3. Select either Mail Recipient as Link or Mail Recipient as Attachment.

My Workspace

My Workspace in GCdocs is equivalent to your personal drive. It is a space where you can save professional personal documents, bookmarks to documents in GCdocs and other items useful to you.

NOTE: Refer to the IM policy on Policy on Acceptable Network and Device Use and Standard for the Storage of Electronic Information.

To access My Workspace

  1. Choose My Workspace on the Personal menu.

To add Favourites to My Workspace

  1. Click the item’s Functions icon and choose Add to Favorites.

To remove Favourites from My Workspace

  1. Click the item’s Functions icon and choose Remove Favorite.

Searching in GCdocs

The Search box appears on almost every page in GCdocs. It will search for information based on where you are in the folder.

Example:

  • A search for the Enterprise level will look for information throughout GCdocs.
  • A search from the Communications folder will look for information in that folder and in all subfolders.

Exception:

  • A search by GCdocs document number using Search GCdocs will retrieve the document regardless of where you are in the folder structure.

To find information using Search GCdocs

  1. Enter your search terms in the Search box.
  2. Click the Perform Search icon.

To find information using Advanced Search

  1. Choose Search on the Tools menu
  2. In the Full Text section, select the appropriate options in Look For, Modifier and Within.
  3. Enter your search terms in the Search box.
  4. If needed, choose additional Advanced Options.
  5. Click the Search button.

Reserving a Document

The reserve function allows you to check out a document, which prevents others from modifying it.

To Reserve an item

  1. Next to the document name, click the down arrow.
  2. Select Reserve.
  3. Click the Submit button to reserve the document. (A checkmark appears on the reserved item)

NOTES: A document is also automatically reserved when opened in Edit mode from GCdocs.

Also, If you want to work on the document remotely, click the Download link in the Option field. (The option to download exists. Users will have to manually add a version later.)

Favorites

Favorites is a GCdocs features which allows you to choose the folders you use most often within your Sector’s folder structure, and display them in a simple list without necessarily having to drill down through the structure every time. You may also favorite a single file.

To set your Favorites

  1. Click on the Personal tab at the top of the page.
  2. In the drop-down menu, click Favorites, which will take you to the My Favorites page.
  3. Click Add New Favorite in the top right corner.
  4. A pop-up will appear with the folder structure.
  5. If you would like to select one of the listed folders for your favorites, click Select >.
  6. If you want to drill-down into a sub-folder, click on the folder name, keep doing this until you find the folder you want, then click Select >.
  7. Repeat as necessary to add as many files/folders as you want.

To add a Favorites tab

  1. Click on the Personal tab at the top of the page.
  2. In the drop-down menu, click Favorites, which will take you to the My Favorites page.
  3. Click on the Configure Tabs in the top right corner.
  4. Click the Add Tabs in the top right corner.
  5. Name your new Favorites Tab and click Submit.
  6. You have just added a new Favorites Tab.
  7. Repeat as necessary to create more tabs

To organize your Favorites

  1. Once you’re in Favorites, click Edit / Organize.
  2. Drop-down menus will appear next to each folder, allowing you to choose under which tab you would like your Favorite folder displayed.
  3. Once you have selected a location for each folder, click Submit.

To remove Favorites

  1. Once you’re in Favorites, click Edit / Organize.
  2. Beside each folder on the right you will see a checkbox titled Removed.
  3. Select the checkbox for the folder(s) you no longer want in your Favorites.
  4. Click Submit.
  5. You have now removed a Favorite.

My Profile

What your GCdocs profile does:

  • You can view your own and other users’ contact information
  • You can update your profile and tell everyone about yourself
  • You can upload a photo

To update your profile

  1. Click on the Personal tab at the top of the screen. A drop-down menu will appear.
  2. Click on My Profile, which will take you to your profile general page where you will see your own contact information.
  3. Click on the Extended tab on the top of the screen.
  4. In this tab you can type a Biography, Job Description, Past Positions and / or any fields you feel comfortable sharing on GCdocs.
  5. In External Profiles you can link your other accounts such as Facebook, LinkedIn and Twitter.
  6. Click Updated at the bottom of the screen to save any changes.

To add / update your Profile Photo

  1. Click on the Personal tab at the top of the screen. A drop-down menu will appear.
  2. Click My Profile, which will take you to the My Profile main page.
  3. Click on the Personal tab.
  4. Click on the Browse Content Server, find the photo you wish to use and click Enter. (Photo must already have been saved within GCdocs)
  5. Click Update. Your profile will be reloaded with your new photo.

Cross Reference (Xreference)

A Cross Reference (Xreference) links two (or more) GCdocs documents. (Example: A presentation in different languages.)

To create a Cross Reference

  1. Click the Functions icon for a managed object, choose Properties, and then choose XReference.
  2. On the XReference tab of the object’s Properties page, click New Cross-Reference in the Add: Cross-Reference drop-down list.
  3. Click the Browse Content Server button, browse for the Content Server object you want to cross reference, and then click the Select link for the object.
  4. On the Add Cross-Reference page, click a Cross-Reference type in the Cross-Reference Type drop-down list.
  5. Click the Add button.

Notifications

GCdocs will notify you when

  • a new version of a document you contributed to is added.

To add a Notification

  1. Click on the Functions drop-down for a file or folder that you like to be notified on.
  2. Choose Set Notification and choose which type of notification you would like.
  3. Click Submit.

To change your Notification interests

  1. Click on Personal at the top of the screen.
  2. In the drop-down menu, click Notification.
  3. On the right side of the screen, click Modify Interests.
  4. Use the drop-down menu to add events to your notification list. Choose different report numbers if you want to receive notification for different events at varying times.
  5. Click Submit.

To change your Notification settings

  1. Click on Personal at the top of the screen.
  2. In the drop-down menu, click Notification.
  3. On the right side of the screen, click Modify Interests.
  4. Use the drop-down menu to add events to your notification list. Choose different report numbers if you want to receive notification for different events at varying times.
  5. Click Submit.
  6. Click on Modify Settings.
  7. A new page will appear with your current settings.
  8. If you would like to receive E-mail notifications, ensure that the checkbox entitled Enable E-mail Delivery of the Report is selected, and that your E-mail address appears in the appropriate field.
  9. Select which days you would like to receive notifications. (Recommended that you select the days you are working.)
  10. Select which hours you get notifications and the frequency.
  11. Click Submit.

Password

To reset the GCdocs password

  1. Click on Reset Password link from your E-mail OR
  2. Click the Reset Password on the GCdocs home page
  3. Enter your GCdocs User name (NOTE: This is case sensitive)
  4. Click Submit
  5. This will bring you to the Password Reset page
  6. Enter in your New Password and Verify Password
  7. Enter the Security Code that you would have received in an E-mail
  8. Click Submit

To change the GCdocs password

  1. In the menu bar, click Tools
  2. In the drop-down menu, click Settings
  3. This will bring you to the General Tab of your General Settings
  4. Click the Password tab
  5. Enter your Current Password
  6. Enter your New Password and Verify Password
  7. Click Update

Migration

To export any type of OneNote file (Page, Section or Notebook)

  1. Go to the area you wish to export to GCDocs
  2. Go to OneNote. Click on File, then select Export.
  3. Choose Page, Section, or Notebook and select the Format.
  4. Click on the Export button, the Save As window opens.
  5. Select the location and give it a meaningful name. Click Save.
  6. Save to GCDocs.
  7. Delete locally saved copy from locally saved location

NOTE: Tabs and Notebooks can be saved in the same way by selecting in the previous steps.

To export any type of OneNote file (Pages Only)

  1. Ensure that OneNote is currently displaying the Page you wish to add to GCDocs
  2. Click the File tab, select Send, then Send as a PDF
  3. An E-mail will open with a PDF in it
  4. Open GCDocs Enterprise Connect view and Drag the PDF attachment from the E-mail into the corresponding folder in the GCDocs structure, saving as a new document or a new version.
  5. Close E-mail.

NOTE: This method only works for pages but saves you having to save to and delete a copy locally.
